# Online Retail Data Clustering

This project demonstrates a clustering analysis on an online retail dataset. The notebook performs data preprocessing, feature engineering, and clustering to segment customers based on purchasing behaviors.


## Requirements

To run this project, you need to have the following packages installed. You can use the versions specified below for consistency and reproducibility:

- **Python** >= 3.8
- **Jupyter Notebook** >= 6.0
- **Pandas** >= 1.3
- **NumPy** >= 1.21
- **Matplotlib** >= 3.4
- **Seaborn** >= 0.11
- **Scikit-learn** >= 0.24
- **joblib** (for saving and loading `.pkl` files) >= 1.0


You can install these packages with the following command:

pip install -r requirements.txt


Create a requirements.txt file with the following content to set up the environment:
pandas==1.3.3
numpy==1.21.2
matplotlib==3.4.3
seaborn==0.11.2
scikit-learn==0.24.2
joblib==1.0.1
jupyter==1.0.0



#Run Instructions:

1. git clone <repository-url>
cd <project-directory>

2. Install Required Packages: Use the provided requirements.txt file to install the required dependencies.

pip install -r requirements.txt

3. Launch Jupyter Notebook: Open the notebook in Jupyter.

jupyter notebook

Run the Notebook: Open online_retail_data_clustering (1).ipynb in Jupyter Notebook and run each cell sequentially to execute the code.
